1. INTRODUCTION                                         *pairup-introduction*

Pairup.nvim is a Neovim plugin for inline AI pair programming. Write `cc:`
markers anywhere in your code, save, and Claude edits the file directly.

Example: >
    -- cc: add error handling here
    -- uu: Should I use pcall or assert?
    function process(data)
      return data.value
    end

Save -> Claude reads the file -> executes the instruction -> removes the marker.

When Claude needs more information, it adds `uu:` markers and you can continue
the discussion by appending `cc:` in response.

==============================================================================
2. REQUIREMENTS                                         *pairup-requirements*

- Neovim 0.11+
- Claude Code CLI (https://docs.anthropic.com/en/docs/claude-code)

==============================================================================
3. INSTALLATION                                         *pairup-installation*

Using lazy.nvim: >
    {
      "Piotr1215/pairup.nvim",
      cmd = { "Pairup" },
      keys = {
        { "<leader>cc", "<cmd>Pairup start<cr>", desc = "Start Claude" },
        { "<leader>ct", "<cmd>Pairup toggle<cr>", desc = "Toggle terminal" },
        { "<leader>cq", "<cmd>Pairup questions<cr>", desc = "Show questions" },
        { "<leader>cx", "<cmd>Pairup stop<cr>", desc = "Stop Claude" },
      },
      config = function()
        require("pairup").setup({
          -- Default config works out of the box
        })
      end,
    }

4. CONFIGURATION                                       *pairup-configuration*

The `--permission-mode acceptEdits` flag is included by default.

All settings below are defaults: >
    require("pairup").setup({
      provider = "claude",
      providers = {
        claude = {
          -- Full command with flags (default includes acceptEdits)
          cmd = "claude --permission-mode acceptEdits",
        },
      },
      terminal = {
        split_position = "left",
        split_width = 0.4,
        auto_insert = false,
      },
      auto_refresh = {
        enabled = true,
        interval_ms = 500,
      },
      inline = {
        markers = {
          command = "cc:",
          question = "uu:",
        },
        quickfix = true,
      },
      statusline = {
        auto_inject = true,
      },
      -- Progress bar (optional, disabled by default)
      -- When enabled, add --add-dir /tmp to your command
      progress = {
        enabled = false,
        file = "/tmp/claude_progress",
      },
      operator = {
        key = "gC",
      },
    })

==============================================================================
5. COMMANDS                                                 *pairup-commands*

:Pairup start                                                  *:Pairup-start*
    Start Claude (hidden terminal).

:Pairup stop                                                    *:Pairup-stop*
    Stop Claude.

:Pairup toggle                                                *:Pairup-toggle*
    Show/hide terminal.

:Pairup say {msg}                                                *:Pairup-say*
    Send message to Claude.

:Pairup questions                                          *:Pairup-questions*
    Show `uu:` markers in quickfix.

:Pairup inline                                                *:Pairup-inline*
    Manual cc: trigger.

:Pairup diff                                                    *:Pairup-diff*
    Send git diff to Claude.

:Pairup lsp                                                      *:Pairup-lsp*
    Send LSP diagnostics to Claude.

==============================================================================
6. OPERATOR                                                 *pairup-operator*

Use `gC` to insert cc: markers with proper comment syntax for any filetype.

gCC                     Insert marker above current line (scope: <line>)
gCip / gCap             Insert marker for paragraph (scope: <paragraph>)
gCiw / gCaw             Insert marker for word (scope: <word>, captures text)
gCis / gCas             Insert marker for sentence (scope: <sentence>)
gCi} / gCa}             Insert marker for block (scope: <block>)
gCif / gCaf             Insert marker for function (scope: <function>)
gC (visual)             Insert marker with selected text (scope: <selection>)

Scope hints tell Claude what the instruction applies to: >
    -- cc: <paragraph> refactor this section
    -- cc: <line> add error handling
    -- cc: <selection> some_variable <- rename to camelCase

==============================================================================
7. SIGNS                                                       *pairup-signs*

Markers show in the gutter:
- 󰭻 (yellow) - `cc:` command marker
- 󰞋 (blue) - `uu:` question marker

Navigation:
]C                      Jump to next cc:/uu: marker
[C                      Jump to previous cc:/uu: marker

==============================================================================
8. STATUSLINE                                             *pairup-statusline*

Automatically injected into lualine (or native statusline if no lualine).

- [C]           Claude running
- [C:pending]   Waiting for Claude
- [C:██░░░░░░░░] Progress bar
- [C:ready]     Task complete

To disable auto-inject: >
    require("pairup").setup({ statusline = { auto_inject = false } })

    -- Add to native statusline manually
    vim.o.statusline = '%f %m%=%{g:pairup_indicator} %l:%c'

==============================================================================
9. PLUG MAPPINGS                                       *pairup-plug-mappings*

Available <Plug> mappings for custom keybindings: >
    vim.keymap.set('n', '<leader>cc', '<Plug>(pairup-toggle-session)')
    vim.keymap.set('n', '<leader>ct', '<Plug>(pairup-toggle)')
    vim.keymap.set('n', '<leader>cl', '<Plug>(pairup-lsp)')
    vim.keymap.set('n', '<leader>cd', '<Plug>(pairup-diff)')
    vim.keymap.set('n', '<leader>cq', '<Plug>(pairup-questions)')
    vim.keymap.set('n', '<leader>ci', '<Plug>(pairup-inline)')
    vim.keymap.set('n', ']C', '<Plug>(pairup-next-marker)')
    vim.keymap.set('n', '[C', '<Plug>(pairup-prev-marker)')
