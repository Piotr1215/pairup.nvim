# pairup.nvim

**Real-time AI pair programming for Neovim**

## Features

- Git diff streaming
- Virtual text overlays
- RPC control

## Installation

```lua
{
  "Piotr1215/pairup.nvim",
  config = function()
    require("pairup").setup({})
  end,
}
```

## Usage

Start with `:PairupStart`
